# 1. Introduction:
## What is Git and GitHub?
Git is a local version control system that helps us to track changes in our code. On the other hand, GitHub is a cloud platform where we can store and share Git repositories online.     

Note: GitLab, Bitbucket etc are the alternatives to GitHub.
